<h1>Graphene</h1>

## About Graphene

Graphene is a fully featured 'Low Code Development' and 'Content Management' Platform built on top of Laravel 5.

### Videos / Tutorials
[Introduction to Graphene](https://www.youtube.com/watch?v=qaTPaJ3DFmM)

## Dependencies
 - MySQL 5.7.8 or Later
 - PHP 7.1 or Later 
 - Composer (Latest)
